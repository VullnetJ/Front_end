What is javascript and why I am refering particularly to it.

JavaScript is a programming language of web and HTML.

It is found in most every dynamic websites

It has many libraries and a long history and strong comunity supporting different JavaScript Libraries

To learn more about JavaScript you can find more examples and explanations from Mozilla MDN.


How I learn JavaScript:

To learn JavaScript I usually follow these steps:

  1. Read course materials provided in school.
  2. Search in Google and read different sources.
  3. Read online books from library.
  4. See videos about JavaScript programming.
  5. Use different sources and plafroms especially dedicated only for JavaScript

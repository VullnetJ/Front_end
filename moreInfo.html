Visibility of System Status:

We should know what is going on inside the system.

Matching between the system and real world:

For example to have "Sign Up" button insdead of other ambigious words

User has to have Control and Freedom:

The user must have freedom do undo any accidental actions.

Consistency and Standards:

A submit button should look the same on all pages



More

Error Preventation:

For instance, Google Search trying to correct spelling

Flexibility and Efficiency of use:

The interface should be flexible. There should be the option to have default installation or custom installation

Aesthetic and minimalist design:

The information which is presented should be relevant. Has to be also useful.
